# תוצאות

> תוצרי הפרויקט - מטיוטה ראשונה ועד גרסה סופית.

## מבנה התיקייה

- `drafts/` - גרסאות עבודה, טיוטות, לא לשימוש סופי
- `final/` - גרסאות מאושרות וסופיות בלבד

## כלל עבודה מומלץ

תוצר עובר מ-`drafts/` ל-`final/` רק אחרי אישור מפורש (שלכם, או של בעל העניין הרלוונטי). אל תדרסו קבצים ב-`final/` - תעדו גרסה חדשה ב-`version-history.md`.
