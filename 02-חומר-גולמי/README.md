# חומר גולמי

> כל מה שמזין את הפרויקט אבל הוא לא בעצמו תוצר - מקורות, השראה, ידע רקע.

## מבנה התיקייה

- `references/` - מסמכים, מאמרים, נתונים שמשמשים כבסיס ידע (PDF, קישורים מתועדים, וכו')
- `examples/` - דוגמאות לתוצרים דומים, בין אם כהשראה ובין אם כ"מה לא לעשות"
- `research-notes.md` - הערות מחקר חופשיות

## טיפ

כשמוסיפים קובץ מקור חדש, כדאי לציין במשפט אחד למה הוא רלוונטי - זה חוסך זמן לסוכן שיקרא את זה בעתיד.
