# קוד הפרויקט

> תיקייה זו מיועדת לפרויקטים שכוללים רכיב פיתוח. אם זהו repo, אפשר גם לקשר אליו (symlink) במקום להעתיק קבצים פיזית.

## הערה לעבודה עם Claude Code

כשעובדים מתוך Claude Code על התיקייה הזו, מומלץ ליצור כאן (או בשורש הפרויקט) קובץ `CLAUDE.md` ייעודי לקוד, נפרד מהסוכן המלווה הכללי - שיתאר: ארכיטקטורה, מוסכמות קוד, פקודות build/test נפוצות.
