# כלים

> Prompts, Skills, וקוד - כל מה שמשמש "מנוע" לייצור התוצרים.

## מבנה התיקייה

- `prompts/` - תבניות prompts שחוזרות על עצמן בפרויקט הזה
- `skills/` - הוראות מורחבות לסוכנים (אם נדרשות מעבר ל-`01-צוות-וסוכנים/`)
- `code/` - קוד הפרויקט, אם זהו פרויקט קוד. ראו `tools-checklist.md` לפרטי הסביבה.

## מתי להוסיף קובץ כאן ומתי ב-01-צוות-וסוכנים?

- **01-צוות-וסוכנים** = *מי* עושה את העבודה (תפקיד, גבולות, סטטוס)
- **03-כלים** = *איך* העבודה נעשית בפועל (ה-prompt המדויק, הקוד, הסקריפט)

סוכן יכול להפנות מתוך ה-`instructions.md` שלו לקובץ ספציפי כאן.
