# יומן העברות (Handoff Log)

> כל פעם שתוצר/משימה עוברים מסוכן אחד לאחר, מתעדים כאן שורה. זה נותן "שרשרת אספקה" שאפשר לעקוב אחריה.

| תאריך | מ-(סוכן) | אל-(סוכן) | מה הועבר | קישור לתוצר |
|---|---|---|---|---|
| | | | | |

## תבנית לשורה חדשה (להעתקה)

```
| [תאריך] | [סוכן מקור] | [סוכן יעד] | [תיאור קצר] | [נתיב לקובץ] |
```
