# מעקב זמנים - הסבר

> קבצי ה-Excel בתיקייה זו (`time-log.xlsx`, `estimates-vs-actual.xlsx`) ייבנו בשלב הבא של התבנית.
> קובץ זה מסביר את העקרונות עד אז, ומשמש גם לתיעוד טקסטואלי חופשי במידת הצורך.

## מה עוקבים אחריו

- זמן בפועל לכל משימה/סוכן (בשעות או בסבבי עבודה)
- הערכה מקורית מול בפועל
- חריגות משמעותיות ולמה קרו

## למה זה שימושי

1. **תכנון עתידי מדויק יותר** - בפעם הבאה שיש משימה דומה, יש בסיס נתונים אמיתי להערכה.
2. **זיהוי צווארי בקבוק** - אם סוכן/שלב מסוים תמיד לוקח הרבה יותר זמן מהמתוכנן, זה סימן לבדוק למה.
3. **הזנה ל-05-חוקים-ותובנות** - דפוסי זמן חוזרים הם תובנה לכשעצמה.
